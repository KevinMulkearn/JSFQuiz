Accessing managed beans properties and methods:

A Managed Bean is a regular Java Bean registered and managed by JSF framework.

Managed Beans usually contains getter and setter methods, business logic, and/or HTML form values.

The Managed Bean works as a Model for UI components.

The Managed Bean Can be directly accessed from JSF pages through several navigation methods.

The annotation @ManagedBean defines bean as being managed:

  • Attribute name used to give specific name to a managed bean (class name is used as default).
  • Attribute eager set to true ensures bean is created before first request method.
