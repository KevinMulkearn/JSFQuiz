Creating Data Tables:

The tag <h:dataTable> is used to represent data collections in tabular form.

The user only needs to specify each columns' information, the code automatically iterates through all elements to fill each row.

Elements within <h:dataTable> can be made active, active methods can be used to invoke procedures such as sorting, editing, and deleting.

Attribute value of <h:dataTable> indicates collection of data.

The basic structure of a dataTable is as follows:

<h:dataTable value= # {someBean.collection} var=ref>
   <h:column><f:facet name=“header”>title</facet>
      # {ref.someattrib}</h:column>
   <h:column>...</h:column>
</h:dataTable>
