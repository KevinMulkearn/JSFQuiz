Topic 4

Different Types of Validators:

The four possible JSF validator tags:

  • f:validateLength
  • f:validateLongRange
  • f:validateDoubleRange
  • f:validateRegex

f:validateLength
This tag is used to validate the length of a string value in a particular range.

f:validateLongRange
This tag is used to validate a long integer value in a particular range.

f:validateDoubleRange
This tag is used to validate a value to a range of float values.

f:validateRegex
This tag is used to validate a string value to a required format or according to regular expression.



Inbuilt validators are used to validate length and type of input fields.

It is possible for a user to create custom validators.

Length/Range have "minimum" and "maximum" attribute.

Regex has "pattern" attribute holding regular expression as described in the java.util.regex.Pattern class.

Validators are part of JSF core tag library.
